Стурктура каталогов проекта:
  - Data_subsets - содержит файл с обучающим и тестовым наборами данных для проектирования нейронной сети
  - Object_params_identification - содержит скрипты для создания нейронной сети для идентификации параметров
 модели объекта управления и проверки результатов ее работы
  - TF_params_identification - содержит скрипты для создания нейронной сети для идентификации коэффициентов 
передаточной функции объекта управления и проверки результатов ее работы

Описание скриптов в корневом каталоге:
  - create_training_set.m - скрипт для создания обучающего и тестового наборов данных для проектирования нейронной сети. 
Должен быть выполнен самым первым в начале работы
  - compare_tuning_accuracy.m - скрипт для сравнения точности настройки системы управления с помощью идентификации двух
типов параметров и отображения уровня шума. Нужно выполнить после выполнения скриптов compare_PRs.m в обеих папках

Описание скриптов для создания нейронной сети и проверки результатов ее работы:
  - neuro_net_creating_newrb.m - скрипт для создания нейронной сети. Нужно выполнить следующим после создания 
обучающего и тестового наборов данных
  - neuro_net_testing.m - скрипты выполняет регрессионный анализ на тестовом наборе данных и рассчитывает отклонения 
идентифицируемых параметров относительно проверяемых значений из тестового набора данных
  - compare_PRs.m - скрипт выполняет расчет среднеквадратичного отклонения переходной характеристики системы 
управления после настройки относительно переходной характеристики системы управления с номинальными параметрами
 модели объекта управления и коэффициентами ПР для оценки точности настройки системы. Помехоустойчивость можно проверить,
изменив значение переменной noise_level
  - test_PR.m - скрипт для проверки настройки системы управления с использованием случайного вектора из тестового набора 
данных
  - compare_steps.m - скрипт для сравнения переходных характеристик системы управления до и после настройки системы
с использованием случайного вектора из тестового набора данных
  - calc_params_sensivity.m - скрипт для расчета чувствительности переходной характеристики к изменениям идентифицируемых
параметров 
  - recalc_training_set.m - скрипт для перерасчета вектора параметров механической части объекта управления из наборов данных 
в вектор желаемых значений коэффициентов полиномов передаточной функции

Описание функций:
  - immse.m - функция для расчета среднеквадратичного отклонения между двумя массивами данных
  - decimate.m - функция для децимации вектора данных переходной характеристики системы управления
  - calc_PR.m - функция для расчета коэффициентов полиномиального регулятора
  - calcPolyCoeffs.m, calcPolyABCoeffs.m - функции для расчета коэффициентов полиномов передаточной функции объекта
управления
